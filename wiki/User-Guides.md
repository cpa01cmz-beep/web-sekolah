# User Guides

## Student Portal User Guide

### Dashboard
The student dashboard provides an overview of your schedule, recent announcements, and quick access to your grades and digital ID.

### Schedule
View your class schedule, including subject, teacher, and time information.

### Grades
Access your current grades for all subjects and terms.

### Digital ID
View and download your digital student ID card.

### Announcements
Stay updated with school and class announcements.

## Teacher Portal User Guide

### Dashboard
The teacher dashboard provides an overview of your classes, recent announcements, and quick access to grade submission.

### Class Management
View and manage your assigned classes, including student rosters.

### Grade Submission
Submit and manage student grades for your classes.

### Announcements
Post announcements to your classes.

## Parent Portal User Guide

### Dashboard
The parent dashboard provides an overview of your child's progress and recent announcements.

### Progress Tracking
View your child's grades, attendance, and other academic information.

### Communication
Communicate with your child's teachers through the messaging system.

## Admin Portal User Guide

### Dashboard
The admin dashboard provides an overview of school-wide data and system status.

### User Management
Manage user accounts, including creating, updating, and deleting users.

### System Configuration
Configure system-wide settings and preferences.

### Analytics
View school-wide analytics and reports.

## General Navigation

### Sidebar Menu
Use the sidebar menu to navigate between different sections of the portal.

### User Profile
Access your profile settings and logout option through the user menu in the top right corner.

### Notifications
View system notifications and announcements through the notification bell icon.

## Support

For technical support, please contact the school IT department.
