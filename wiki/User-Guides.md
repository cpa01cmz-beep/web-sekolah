# User Guides

## Student Portal User Guide

### Dashboard
The student dashboard provides an overview of your schedule, recent announcements, and quick access to your grades and digital ID.

### View Your Class Schedule

1. Click on "Schedule" in the sidebar menu
2. Your weekly schedule displays with day, time, subject, and teacher information
3. Use the filter dropdown to show/hide specific days

### Check Your Grades

1. Click on "Grades" in the sidebar menu
2. Your grades display with subject, score, feedback, and submission date
3. Grade colors indicate performance level (green: A, yellow: B, red: C or lower)
4. Click on any grade row to view detailed feedback from teacher

### Download Your Digital Student ID

1. Click on "Student Card" in the sidebar menu
2. Your digital ID card displays with your photo, name, and student ID number
3. Click "Download PDF" to save a copy of your student card
4. The PDF includes a QR code for verification

### View Announcements

1. Announcements display on your dashboard
2. Click any announcement to read full details
3. Filter announcements by date or search by keyword
4. Recent announcements show notification badge

## Teacher Portal User Guide

### Dashboard
The teacher dashboard provides an overview of your classes, recent announcements, and quick access to grade submission.

### View Your Classes

1. Click on "Dashboard" in the sidebar menu
2. Your classes display with student count and recent activity
3. Click on any class to view student roster

### Submit Grades for a Student

1. Click on "Classes" in the sidebar menu
2. Select the class you want to grade
3. The student list displays with current grades
4. Find the student you want to grade
5. Click the "Edit" button in the grade column
6. Enter the score (0-100) and optional feedback
7. Click "Save Changes" to submit the grade

**Important**: Grade must be between 0 and 100. Invalid scores will be rejected.

### Create an Announcement

1. Click on "Announcements" in the sidebar menu
2. Click "New Announcement" button
3. Enter announcement title and content
4. Select target audience (specific class or all students)
5. Click "Publish" to post the announcement

## Parent Portal User Guide

### Dashboard
The parent dashboard provides an overview of your child's progress and recent announcements.

### View Your Child's Grades

1. Log in with parent account credentials
2. Your child's grades display on the dashboard
3. Click on "Grades" in the sidebar for detailed view
4. Grades show subject, score, teacher feedback, and date

### Check Child's Schedule

1. Click on "Schedule" in the sidebar menu
2. Your child's weekly schedule displays
3. View class times, subjects, and teachers
4. Contact teacher information available for each subject

## Admin Portal User Guide

### Dashboard
The admin dashboard provides an overview of school-wide data and system status.

### Add a New User

1. Click on "User Management" in the sidebar menu
2. Click "Add User" button
3. Fill in required fields:
   - Name
   - Email
   - Role (Student, Teacher, Parent, Admin)
   - For students: Class ID and Student ID Number
   - For teachers: Class IDs they teach
   - For parents: Child's Student ID
4. Click "Create User" to add the account

### Edit User Information

1. Go to "User Management"
2. Find the user you want to edit
3. Click the "Edit" button in the Actions column
4. Update the fields you need to change
5. Click "Save Changes" to update the user

### Delete a User

1. Go to "User Management"
2. Find the user you want to delete
3. Click the "Delete" button in the Actions column
4. Confirm deletion in the dialog

**Note**: Users with associated data (teachers with classes, students with grades) will show warnings before deletion can proceed.

### View System Statistics

1. Click on "Dashboard" in the sidebar menu
2. View enrollment statistics, user counts, and activity metrics
3. Charts display trends over time
4. Filter by date range to view specific periods

## Authentication

### Login Process
1. Navigate to the login page
2. Enter your email and password
3. Select your role (Student, Teacher, Parent, or Admin)
4. Click the login button for your role
5. You will be redirected to your role-specific dashboard

### Logout Process
1. Click on your profile icon in the top right corner
2. Select "Logout" from the dropdown menu
3. You will be redirected to the login page

## Troubleshooting

### Login Issues

**Problem**: Cannot log in or receive "Invalid credentials" error
- **Solution**: Verify your email and password are correct. Check your role selection matches your assigned role.
- **Contact**: IT support if issue persists

**Problem**: Redirected to wrong portal after login
- **Solution**: Ensure you selected the correct role (Student, Teacher, Parent, Admin) before logging in.

### Data Not Loading

**Problem**: Dashboard or grades page shows loading spinner indefinitely
- **Solution**: Refresh the page by pressing F5 or clicking the browser refresh button.
- **Check**: Internet connection is stable.
- **Try**: Clear browser cache and cookies if issue persists.

### Grade Submission Errors (Teachers)

**Problem**: Cannot submit grade, shows validation error
- **Solution**: Ensure score is between 0 and 100. Check that both score and course are selected.

**Problem**: Grade saves but doesn't appear in list
- **Solution**: Refresh the page to update the display. Data may be cached.

### PDF Download Issues (Students)

**Problem**: Student card PDF doesn't download
- **Solution**: Check browser's download folder. Some browsers block automatic downloads.
- **Enable**: Allow downloads in browser settings if prompted.

### Performance Issues

**Problem**: Application feels slow or laggy
- **Solution**: Close other browser tabs. Clear browser cache. Try using a different browser (Chrome, Firefox, Safari).

### Mobile Access

**Problem**: Sidebar menu not accessible on mobile
- **Solution**: Tap the hamburger menu icon (three horizontal lines) in the top left corner to open the sidebar.

**Problem**: Tables are hard to read on mobile
- **Solution**: Scroll horizontally to view all table columns. The application is optimized for desktop but works on mobile devices.

### Reporting Issues

When reporting a technical issue, please include:
- Your email address
- The role you're using (Student, Teacher, Parent, Admin)
- A screenshot of the error (if applicable)
- Steps you took before encountering the issue
- Browser and version information
